# Blackjack ♠️
A simplified Blackjack game built with Python.

## What it does

* Deals two random cards each to the player and computer at the start
* Lets the player draw one additional card if they choose
* Checks for busts, wins, losses, and draws based on card totals
* Reveals the computer's first card only, keeping the second hidden

## What I learned

* Random number generation
* List manipulation and sum functions
* Conditional logic with chained comparisons
* Structuring game logic into functions

## How to run

1. Clone the repository
2. Run `python Blackjack.py`
3. Choose to play, then decide whether to draw an extra card
